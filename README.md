# MediPredict AI

An AI-powered health insurance analytics platform built using Streamlit and Machine Learning.

## Features

- Risk Prediction
- Premium Prediction
- Fraud Detection
- Claim Amount Prediction
- Batch CSV Prediction
- Explainable AI Insights

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- XGBoost
- Optuna
- Pandas
- NumPy
